# Global-Superstore-Project-Power-BI-

This interactive Power BI dashboard visualizes global sales performance across categories, regions, years, and shipping modes. It provides insights into sales growth trends, category-wise performance, and profitability using filters for category, priority, and ship mode.

 Key Insights:

Technology is the top-selling category with 4.7M in sales.

Consistent YoY growth, reaching 4.3M sales in 2014.

Medium-priority orders contribute the highest revenue (7.3M).

Standard Class shipping accounts for ~60% of all orders.

Profitability increased steadily from 0.2M to 0.5M over 4 years.
